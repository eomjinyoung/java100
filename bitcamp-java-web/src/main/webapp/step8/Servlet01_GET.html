GET 요청의 예

0. 웹브라우저에 URL 입력

서버에 GET 요청을 보내는 사례를 알아보자! GET 요청은 항상 URL에 값이 붙는다.

"http://localhost:9999/java-web/step8/Servlet01?name=홍길동&age=20" 과 같은 URL 주소를 웹브라우저에 입력하면 웹 브라우저는 무조건 GET 요청을 보낸다.

1. 링크

링크를 누르면 웹브라우저는 서버에 GET 요청을 보낸다.

2. 폼(method 속성 생략)

form 태그는 입력양식을 만들 때 주로 사용한다. 즉 사용자로부터 특정한 값을 받고 싶을 때 이 태그를 사용하여 입력 양식을 출력한다. method 속성을 생략하면 기본 값이 'GET'이기 때문에 양식에 포함된 전송 버튼을 누르면 GET 요청을 보내게 된다.
name:
age:
form 태그에 method 속성을 생략하면 기본이 GET이다.

3. 폼(method 속성 추가)

물론 form 태그에 method 속성의 값을 GET으로 명시해도 된다.
name:
age:

4. 자바스크립트: location.href="url"

자바스크립트의 location 객체를 사용하면 GET 요청을 할 수 있다.
